# ericmuckley

This is a static SPA built in [SvelteKit](https://kit.svelte.dev/).

## Developing

Download this folder and run

```bash

# install dependencies
npm install

# start the development server in a new browser tab
npm run dev -- --open

# run code linters
npm run format

# build for deployment
npm run build

```
